rené is a design system for styling clean, simple, lightweight interfaces. it's the first AI-ready CSS framework, creating the perfect bridge between design and code. its utility-first approach makes it naturally compatible with AI design-to-code workflows, while remaining easy enough to "get it" for humans. no weird stuff.

as both a library and framework, rené offers ready-to-use elements and a consistent, customizable structure. it prioritizes utility-first classes yet allows for inline styles, making it ideal for AI interpretation while maintaining clean, maintainable code. this makes rené the ultimate common ground for designers, developers, and AI tools.
